So sánh html {font-size: 62.5%;} và :root {font-size: 62.5%;}

Khuyến nghị

Nên sử dụng :root { font-size: 62.5%; } vì:

  • Độ đặc hiệu cao hơn (0,1,0 so với 0,0,1)
  • Phù hợp hơn khi khai báo biến CSS (custom properties)
  • Ít có khả năng bị ghi đè bởi các quy tắc khác

Kết quả tính toán

Với font-size mặc định của trình duyệt là 16px:

Giá trị rem Kích thước thực tế (px) Ví dụ trực quan
1rem 10px (62.5% × 16px)
Văn bản 1rem
1.6rem 16px
Văn bản 1.6rem
2.4rem 24px
Văn bản 2.4rem

Ví dụ mã CSS

Sử dụng :root

:root {
  font-size: 62.5%; /* 1rem = 10px */
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
}

body {
  font-size: 1.6rem; /* 16px */
}

h1 {
  font-size: 3.2rem; /* 32px */
  color: var(--primary-color);
}

Sử dụng html

html {
  font-size: 62.5%; /* 1rem = 10px */
}

body {
  font-size: 1.6rem; /* 16px */
}

h1 {
  font-size: 3.2rem; /* 32px */
}

Lưu ý quan trọng

  • Kỹ thuật này chỉ hoạt động khi font-size mặc định của trình duyệt là 16px
  • Nếu người dùng thay đổi cài đặt font-size mặc định, tỷ lệ sẽ vẫn được duy trì
  • Luôn kiểm tra hiển thị trên nhiều trình duyệt và thiết bị
